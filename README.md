# Compilar Sass

En este repo están los ejemplos del curso de Sass para las clases de "Compilado de Sass". [Curso de Sass](https://escuela.it/cursos/sass) de EscuelaIT.

- En la rama "[main](https://github.com/EscuelaIt/Compilar-Sass/)" encontramos el código de base que hemos usado para la compilación.
- En la rama "[sass-node-simple](https://github.com/EscuelaIt/Compilar-Sass/tree/sass_node_simple)" encontramos el flujo creado para compilar simplemente con nodejs.
- En la rama "[sass-webpack](https://github.com/EscuelaIt/Compilar-Sass/tree/sass-webpack)" encontramos el flujo para compilar Sass con Webpack.
- En la rama "[compilar-sass-mix](https://github.com/EscuelaIt/Compilar-Sass/tree/compilar-sass-mix)" encontramos el flujo para compilar Sass con Laravel Mix.
- En la rama "[Ejemplos each](https://github.com/EscuelaIt/Compilar-Sass/tree/Ejemplos-each)" encontramos los ejemplos de la clase de recorridos con @each. Esta rama la compilamos con Mix.
