# Compilar Sass

En este repo están los ejemplos del curso de Sass para las clases de "Compilado de Sass". [Curso de Sass](https://escuela.it/cursos/sass) de EscuelaIT.

- En la rama "master" encontramos el código de base que hemos usado para la compilación.
- En la rama "sass-node-simple" encontramos el flujo creado para compilar simplemente con nodejs
- En la rama "sass-webpack" encontramos el flujo para compilar Sass con Webpack
